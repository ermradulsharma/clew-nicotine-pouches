# Security Policy

## Supported Versions

The following versions of Clew Pouches are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please do NOT open a public issue. Instead, please report it via one of the following methods:

- **Email**: Send an email to security@clewpouches.com (placeholder)
- **GitHub**: Use the "Report a security vulnerability" button on the GitHub Security tab if available.

Please include as much detail as possible, including:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge your report within 48 hours and provide updates on our progress in addressing the issue.

## Response Process

1. **Acknowledge**: We will confirm receipt of your report.
2. **Investigate**: We will verify the vulnerability.
3. **Fix**: We will develop a patch.
4. **Notify**: We will release the fix and notify you.
